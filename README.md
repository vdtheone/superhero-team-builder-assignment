# Superhero Team Builder

A full-stack web application that allows users to explore superheroes, view detailed attributes, build balanced teams, and receive intelligent team recommendations based on hero powers. The application integrates with the Superhero API, supports favourite heroes, role-based editing, and team comparison features.

## Features

*   **Hero Directory:** Browse, search, and sort superheroes by name or specific attributes.
*   **Hero Details:** View comprehensive stats including intelligence, strength, speed, durability, power, and combat.
*   **User Accounts:** Secure registration and login functionalities.
*   **Favourites:** Save your favourite superheroes for quick access.
*   **Team Building:** Create, manage, and customize your own superhero rosters.
*   **Team Comparison:** Compare two distinct teams side-by-side to evaluate statistical advantages.
*   **Smart Recommendations:** Receive intelligent suggestions for heroes to add based on your team's current stat distribution.

## Setup Instructions

### Prerequisites
*   Node.js (v18+)
*   Python (3.9+)

### Backend Setup (Django)
1. Navigate to the backend directory.
2. Create a Python virtual environment:
   ```bash
   python -m venv venv
   ```
3. Activate the virtual environment:

   Windows: 
   ```bash
   venv\Scripts\activate
   ```
   Mac/Linux: 
   ```bash
   source venv/bin/activate
   ```
4. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
5. Create a `.env` file in the `backend` directory and add your Superhero API token:
   ```env
   SUPERHERO_API_TOKEN=your_api_token_here
   ```
6. Run the automated startup script:
   ```bash
   chmod +x start.sh
   ./start.sh
   ```
   *Note: This script will apply migrations, create default users (`admin` / `adminpass` and `player1` / `playerpass`), seed superhero data, and start the development server on port 8001.*

### Frontend Setup (React + Vite)
1. Navigate to the `frontend` directory.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the Vite development server:
   ```bash
   npm run dev
   ```

## Core Logic Explanations

### Team Comparison Logic
The team comparison feature evaluates and visualizes the strengths of two distinct teams to determine which roster holds the statistical advantage.
1. **Attribute Aggregation:** For each team, the system calculates the total sum (or average) of all six core attributes (Intelligence, Strength, Speed, Durability, Power, Combat) provided by the heroes currently in that team.
2. **Side-by-Side Analysis:** The aggregated stats of Team A are compared directly against Team B. This makes it easy to quickly spot which team dominates in specific battle categories.
3. **Overall Power Level:** A cumulative "Total Power" score is generated by combining all attributes. The team with the highest overall score is highlighted as the statistical winner, helping users understand match-up advantages.

### Smart Recommendation Logic
The recommendation engine acts as a strategic assistant to help you build the ultimate, well-rounded team.
1. **Weakness Identification:** The system calculates the average stats of your current team lineup. It actively searches for your team's lowest attributes (e.g., a team of pure brawlers might have exceptionally high Strength but very low Intelligence or Speed).
2. **Candidate Filtering:** It scans the database of available superheroes, automatically excluding heroes that are already currently on your roster.
3. **Strategic Matching:** The system scores available heroes based on how well they compensate for your team's specific weaknesses. For example, if your team severely lacks Intelligence, the algorithm will highly prioritize and recommend heroes (like Batman or Iron Man) who possess near-max Intelligence to perfectly balance out your team's overall capabilities.
